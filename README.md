# 🛒 E-Commerce Sales Dashboard

## 📊 Project Overview

This project presents an end-to-end data analytics workflow using E-commerce sales data. The data was explored, cleaned, and analyzed using multiple tools to extract meaningful business insights and build an interactive dashboard.

---

## 🛠 Tools & Technologies Used

* **Excel** → Initial data understanding & quick exploration
* **Python (Pandas, Matplotlib, Seaborn)** → Data cleaning & EDA
* **SQL Server** → Data analysis & business queries
* **Power BI** → Interactive dashboard & visualization

---

## 🔄 Project Workflow

1. **Excel**

   * Understood dataset structure
   * Reviewed columns, basic patterns

2. **Python**

   * Handled missing values
   * Converted date columns
   * Performed exploratory data analysis (EDA)

3. **SQL Server**

   * Performed business analysis:

     * Sales by Region
     * Profit by Category
     * Loss-making products
     * Discount impact on profit

4. **Power BI**

   * Built interactive dashboard
   * Added advanced features:

     * KPI Cards
     * Combo Chart (Sales vs Profit)
     * Heatmap (Category vs Region)
     * Scatter Plot (Discount vs Profit)
     * Top Customers Table
     * Tooltip (Hover insights)
     * Drill-through (Detailed analysis)

---

## 📈 Key Insights

* West region generates highest sales
* Technology category is most profitable
* Higher discounts reduce profit
* Several products are loss-making
